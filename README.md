# Comparative Analysis of Value at Risk (VaR) for a Portfolio Using Traditional, Statistical, and Deep Learning Methods
## Abstract

This project presents a comparative analysis of portfolio Value at Risk (VaR) estimation using traditional methods, statistical approaches, and deep learning techniques. The objective is to evaluate model performance and risk forecasting capabilities across different methodologies.

## Objectives

- Estimate portfolio VaR
- Compare traditional and statistical methods
- Implement deep learning models
- Evaluate predictive performance
- Analyze portfolio risk behavior

## Methodology

1. Data Collection
2. Portfolio Construction
3. Data Preprocessing
4. Traditional VaR Models
5. Statistical Methods
6. Deep Learning Models
7. Performance Evaluation

## Models Used

### Traditional Methods
- Historical Simulation
- Variance-Covariance Method

### Statistical Methods
- Volatility Models
- Monte Carlo Simulation

### Deep Learning Methods
- LSTM
- Neural Networks
